Detailed Description

template<class LineIterator, class TokenIterator, class Iterator>
class lexer::LexerTemplateIterator< LineIterator, TokenIterator, Iterator >

A template parent class for all types of iterators.

Template Parameters
LineIterator- an iterator that is used to track the current line.
TokenIterator- an iterator that is used to track the current token.
Iterator- child class.
